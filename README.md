## Requirements

The following versions of PHP are supported by this version.

* PHP 5.3
* PHP 5.4
* PHP 5.5
* PHP 5.6

The following versions of PHP are NOT supported by this version.

* PHP 7.0
* HHVM


## Installation / Usage

Via Composer

``` json
{
    "require": {
        "elnebuloso/flex-crypt": "~2.0"
    }
}
```
